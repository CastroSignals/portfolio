Personal portfolio site showcasing 8 years of frontend engineering work, from airline revenue dashboards to logistics platforms.

## Tech Stack

- Angular 21
- TypeScript
- SCSS (BEM methodology)
- D3.js (data visualizations)
- GitHub Pages (deployment)

## Running Locally

```bash
npm install
ng serve
```

Open [http://localhost:4200](http://localhost:4200) in your browser.

## Architecture

Single-page Angular application with standalone components, lazy-loaded route for the projects/timeline page, and a component-based structure under `src/app/components/` (shared UI) and `src/app/pages/` (route-level views). Styling uses SCSS with CSS custom properties for theming.
